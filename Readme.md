 # Nodejs Microservices Boilerplate:

API Gateway folder structure:
```
- Config
- handlers
    - index.js (It will contains all routes)
    - login
        index.js (type of routes)
            post.js
        get.js
- Helpers
- node_modules
- app.js
```
